# Templated Documentation

Some Kyma repositories use the [doctemplater](https://github.tools.sap/kyma/doctemplater) tool to automatically generate documentation from JSON schemas and component-version YAML files. This approach is particularly useful when documentation must dynamically reflect data structures or configurations.

Consider using doctemplater when:
- Documentation must stay synchronized with JSON schemas or data structures that are often updated.
- You need to display component version information dynamically.
- Documentation contains repetitive patterns that can be templated.
- Multiple documentation pages share common structure or data sources.

For more information, see the [doctemplater repository](https://github.tools.sap/kyma/doctemplater).
